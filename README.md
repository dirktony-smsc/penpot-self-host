# penpot-self-host

A collection of _scripts_ to run [penpot](https://penpot.app) on my computer.

## How to use?

1. Download the penpot [`docker-compose.yaml`](https://github.com/penpot/penpot/blob/develop/docker/images/docker-compose.yaml)
by running the `download-penpot-docker-compose.sh` script

2. Start penpot with `start.sh`

3. Stop penpot with `stop.sh`
